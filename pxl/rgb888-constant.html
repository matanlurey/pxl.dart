Rgb888 const rgb888

24-bit^1 RGB pixel format with three 8-bit channels.

Colors in this format are represented as follows:

Color Value
Rgb.red 0xFF0000
Rgb.green 0x00FF00
Rgb.blue 0x0000FF
Rgb.white 0xFFFFFF
Rgb.black 0x000000

^1: The 24-bit RGB format is a 32-bit format with 8 unused bits.
